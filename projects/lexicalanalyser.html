Key Features

Complete C Tokenization

Recognizes all C keywords, identifiers, constants (hex, octal, binary, float), operators, and special characters with 100% accuracy.

Advanced Error Detection

Detects invalid constants, unterminated strings/chars, malformed numbers, and reports line numbers for easy debugging.

Comment Handling

Smart handling of both single-line (//) and multi-line (/* */) comments, including detection of unterminated block comments.

Multi-char Operators

Correctly tokenizes complex operators like ++, --, ==, !=, <=, >=, &&, ||, +=, -=, *=, /=, %=, ->, <<=, >>=, etc.

Supported Token Types

Token Type Examples Detection Method
KEYWORD int, if, while, return, struct String comparison against 32 keywords
IDENTIFIER variable_name, func1, _temp Regex: [a-zA-Z_][a-zA-Z0-9_]*
CONSTANT 123, 0xFF, 0755, 0b1010, 3.14e-10 State machine with validation
OPERATOR +, -, *, /, ==, &&, +=, -> Lookup table with multi-char support
SPECIAL CHARACTER { } ( ) [ ] ; , . Direct character comparison

Core Implementation

int is_constant(char *token, int line_num) {
    int len = strlen(token);
    
    // Hexadecimal
    if(len > 2 && token[0] == '0' && (token[1] == 'x' || token[1] == 'X')) {
        for(int i = 2; i < len; i++) {
            if(!isxdigit(token[i])) {
                printf("ERROR: Invalid hex '%s'\n", token);
                return FAILURE;
            }
        }
        return CONSTANT;
    }
    
    // Floating point validation
    int decimal_count = 0;
    for(int i = 0; i < len; i++) {
        if(token[i] == '.') decimal_count++;
    }
    if(decimal_count > 1) {
        printf("ERROR: Multiple decimals\n");
        return FAILURE;
    }
    
    return CONSTANT;
}
