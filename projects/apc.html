Arbitrary Precision Calculator

A C-based calculator that performs arithmetic operations on integers of unlimited size using doubly linked list data structures. Implements manual addition, subtraction, multiplication, and division algorithms that handle numbers beyond standard data type limits.

Core Features

Unlimited Precision

Handles numbers of any size by storing digits in doubly linked lists, breaking 64-bit integer limits.

Four Operations

Supports addition, subtraction, multiplication, and division with proper sign handling and edge cases.

Grade-School Algorithms

Implements manual arithmetic algorithms digit-by-digit with carry/borrow propagation.

Doubly Linked Lists

Uses custom DLL implementation for efficient bidirectional traversal during arithmetic operations.

Negative Number Support

Properly handles negative operands and calculates correct sign for all operations.

Zero Elimination

Automatically removes leading zeros from results for clean output formatting.

Arithmetic Operations

Operation Algorithm Complexity
Addition • Start from least significant digit (tail)
• Add digits with carry propagation
• Create new nodes for result		 O(max(n,m))
Subtraction • Compare numbers to determine larger
• Subtract digit-by-digit with borrow
• Handle negative results		 O(max(n,m))
Multiplication • Grade-school multiplication algorithm
• Partial products with shifting
• Sum partial results		 O(n×m)
Division • Long division using repeated subtraction
• Compare divisor with dividend segments
• Calculate quotient digit-by-digit		 O(n×m)

Key Challenges & Solutions

Unlimited Size

Challenge: Standard data types limit number size. Solution: Used doubly linked lists to store digits dynamically.

Bidirectional Operations

Challenge: Arithmetic needs both forward and backward traversal. Solution: Implemented DLL for efficient bidirectional access.

Carry/Borrow Propagation

Challenge: Handling carry in addition and borrow in subtraction across thousands of digits. Solution: Implemented manual propagation algorithms.

Compilation & Usage

+ Addition
- Subtraction

/ Division
