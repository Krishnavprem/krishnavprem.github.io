Machine Learning Models

DenseNet (Transfer Learning)

Fine-tuned a pre-trained DenseNet model to extract high-level features from cattle skin images, improving accuracy with limited training data.

Custom CNN

Designed a lightweight convolutional neural network from scratch to compare performance and generalization against transfer learning models.

GoogLeNet (Transfer Learning)

Utilized an ImageNet-pretrained GoogLeNet architecture to perform efficient multi-scale feature extraction with reduced computation.

Web Interface

A simple and intuitive web interface allows farmers to upload cattle skin images and receive disease predictions without requiring technical expertise.
