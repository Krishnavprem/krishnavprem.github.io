Address Book System

A console-based C application for managing personal contacts with full CRUD operations, input validation, and file persistence. Built with modular architecture and comprehensive data validation.

Core Features

Create Contact

Add new contacts with comprehensive validation for name, phone number, and email format.

Search Contact

Search by name (substring), phone number (exact), or email (exact) with case-insensitive matching.

Edit Contact

Update any field of existing contacts with intelligent duplicate detection and validation.

Delete Contact

Remove contacts with confirmation and handling of multiple matches.

List Contacts

Display all contacts in sorted alphabetical order with formatted table output.

File Persistence

Automatic save/load to CSV file with robust error handling and data integrity.

Input Validation

Field Validation Rules
Name • Only letters, spaces, and dots allowed
• Must not start with dot
• Maximum 50 characters
Phone Number • Exactly 10 digits
• Cannot start with 0
• Only numeric characters
• Must be unique
Email • Must contain exactly one '@'
• Must end with '.com'
• Only letters, numbers, underscores, '@', '.' allowed
• Must be unique

Code Architecture

Modular File Structure

main.c - Program entry point and menu system
contact.h/c - Core data structures and operations
file.h/c - File I/O and persistence layer
populate.h/c - Dummy data generation
// Example: Core Contact Structure
#define MAX_CONTACTS 100

typedef struct {
  char name[50];
  char phone[20];
  char email[50];
} Contact;

typedef struct {
  Contact contacts[MAX_CONTACTS];
  int contactCount;
} AddressBook;
